SYSTEM REQUIREMENTS
NODE.JS
Version 18 or later. Download from nodejs.org. Required to run the Electron app.
⬡ REQUIRED
PYTHON 3
Version 3.9+. Usually pre-installed on macOS. Powers the speech recognition engine.
⬡ REQUIRED
HOMEBREW
Package manager for macOS. Needed for PyAudio and optional brightness control. Install at brew.sh.
◇ RECOMMENDED
BRIGHTNESS CLI
Enables "brighter" / "dimmer" voice commands. Install with: brew install brightness
